How to Install

Composer

To install MkyFramework with composer use the command

      composer create-project micky/mky appname --prefer-dist

This command will install the new app in your app name folder. After that you have to config your app in the config folder

Configuration

File config point Description
app app_name Application name
timezone Default timezone value for datetime instance
locale Default location value (useful for translation)
route_mode Default route mode for all application, (controller, file, both)
security.csrf Enable csrf security
debugMode Enable debug mode
database connections.mysql MYSQL connexion data
auth providers Set credentials and manager class for login
remember Set provider remember token lifetime
password_reset Set provider password_reset token lifetime
cache spaces Set file driver and path for cache
layouts Root folder for .mky layouts
includes Root folder for .mky included views
